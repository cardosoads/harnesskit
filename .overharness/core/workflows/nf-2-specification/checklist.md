# Validation Checklist: Feature Stories

---

## Required Items

- [ ] **At least 3 user stories** — Are there at least 3 stories with complete format?
  - Criterion: 3+ stories with As a/I want to/So that and acceptance criteria.

- [ ] **Acceptance criteria per story** — Does every story have at least 2 acceptance criteria?
  - Criterion: every story has 2+ Given/When/Then criteria.

- [ ] **Priority assigned** — Does every story have a MoSCoW priority?
  - Criterion: every story has Must/Should/Could/Won't assigned.

- [ ] **Summary table present** — Is there a summary with all stories at a glance?
  - Criterion: table includes ID, story, feature, priority, effort.

---

## Optional Items (recommended)

- [ ] **Technical notes reference impact analysis** — Do stories reference Raj's findings?
  - Criterion: at least 2 stories have technical notes from the impact analysis.

- [ ] **Effort estimated** — Does every story have an effort estimate?
  - Criterion: every story has S/M/L assigned.

---

## Validation Result

- **All required items approved?** -> Proceed to architecture.
- **Any required item failed?** -> Fix before proceeding.
